# ohdsi-sqlrender

Python wrapper for the OHDSI SqlRender R package.

## Installation

```bash
pip install ohdsi-sqlrender
```

## Usage

```python
from ohdsi.sqlrender import SqlRender

# Initialize the SQL renderer
sr = SqlRender()

# Your code here
```

## Requirements

- Python >= 3.13
- R environment with OHDSI packages installed
- rpy2
- pandas
